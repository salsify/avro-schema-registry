Next Steps

  1. Check the Heroku config for your app to find the value set for the SCHEMA_REGISTRY_PASSWORD. You'll need this value to make requests.
  2. Use a client to make some requests:
    • avro_turf
    • schema_registry
    • curl:

      curl -X POST --data-binary @request.json \
        -H "Content-Type: application/vnd.schemaregistry.v1+json" \
        https://username:SCHEMA_REGISTRY_PASSWORD@\
          avro-schema-registry.example.com/subjects/NAME/versions

  3. Learn about the API in the Confluent Schema Registry documentation.
  4. See the implementation in the avro-schema-registry repo.
  5. Learn more about why we built it on the Salsify blog.
