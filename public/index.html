avro-schema-registry

A service for storing and retrieving versions of Avro schemas

Description

This service implements the API of the Confluent Schema Registry.

Security

This service is public. Uploaded schemas are visible to anyone.

The password for the service is 'avro'. The username for requests is not verified but you should choose something to identify yourself.

Data stored by this service may be cleared at any time.
